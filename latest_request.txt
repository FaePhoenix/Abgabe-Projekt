Im engeren Sinne sind nur die f\u00fcr das menschliche <a href=\"/wiki/Visuelles_System\" title=\"Visuelles System\">Auge</a> <a href=\"/wiki/Farbwahrnehmung#Sichtbares_Licht\" title=\"Farbwahrnehmung\">sichtbaren Anteile</a> des <a href=\"/wiki/Elektromagnetisches_Spektrum\" title=\"Elektromagnetisches Spektrum\">elektromagnetischen Spektrums</a> gemeint. Die untere Grenze dieses Bereichs wird bei Wellenl\u00e4ngen von 360 bis 400 nm, die obere Grenze bei 760 bis 830 nm angesetzt.<sup id=\"cite_ref-1\" class=\"reference\"><a href=\"#cite_note-1\"><span class=\"cite-bracket\">&#91;</span>1<span class=\"cite-bracket\">&#93;</span></a></sup> Im weiteren Sinne werden in der Physik auch elektromagnetische Wellen k\u00fcrzerer <a href=\"/wiki/Wellenl%C3%A4nge\" title=\"Wellenl\u00e4nge\">Wellenl\u00e4nge</a> (<a href=\"/wiki/Ultraviolett\" class=\"mw-redirect\" title=\"Ultraviolett\">Ultraviolett</a> oder <a href=\"/wiki/R%C3%B6ntgenstrahlung\" title=\"R\u00f6ntgenstrahlung\">R\u00f6ntgenstrahlung</a>) und gr\u00f6\u00dferer Wellenl\u00e4nge (<a href=\"/wiki/Infrarot\" class=\"mw-redirect\" title=\"Infrarot\">Infrarot</a> und <a href=\"/wiki/Terahertz-Strahlung\" class=\"mw-redirect\" title=\"Terahertz-Strahlung\">Terahertz-Strahlung</a>) dazu gez\u00e4hlt, wobei die <a href=\"/wiki/Internationale_Beleuchtungskommission\" title=\"Internationale Beleuchtungskommission\">Internationale Beleuchtungskommission</a> von der Verwendung der Bezeichnung Licht in diesem Kontext abr\u00e4t.<sup id=\"cite_ref-2\" class=\"reference\"><a href=\"#cite_note-2\"><span class=\"cite-bracket\">&#91;</span>2<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p><p>Die physikalischen Eigenschaften des Lichts werden durch verschiedene Modelle beschrieben: In der <a href=\"/wiki/Strahlenoptik\" class=\"mw-redirect\" title=\"Strahlenoptik\">Strahlenoptik</a> wird die geradlinige Ausbreitung des Lichts durch \u201eLichtstrahlen\u201c veranschaulicht; in der <a href=\"/wiki/Wellenoptik\" title=\"Wellenoptik\">Wellenoptik</a> wird die Wellennatur des Lichts betont, wodurch auch <a href=\"/wiki/Beugung_(Physik)\" title=\"Beugung (Physik)\">Beugungs</a>- und <a href=\"/wiki/Interferenz_(Physik)\" title=\"Interferenz (Physik)\">Interferenzerscheinungen</a> erkl\u00e4rt werden k\u00f6nnen. In der <a href=\"/wiki/Quantenphysik\" title=\"Quantenphysik\">Quantenphysik</a> schlie\u00dflich wird das Licht als ein Strom von Quantenobjekten, den <a href=\"/wiki/Photon\" title=\"Photon\">Photonen</a> (veranschaulichend auch \u201eLichtteilchen\u201c genannt), beschrieben. Eine vollst\u00e4ndige Beschreibung des Lichts bietet die <a href=\"/wiki/Quantenelektrodynamik\" title=\"Quantenelektrodynamik\">Quantenelektrodynamik</a>. Im <a href=\"/wiki/Vakuum\" title=\"Vakuum\">Vakuum</a> breitet sich Licht mit der konstanten <a href=\"/wiki/Lichtgeschwindigkeit\" title=\"Lichtgeschwindigkeit\">Lichtgeschwindigkeit</a> von 299.792.458&#8239;m/s aus. Trifft Licht auf Materie, so kann es <a href=\"/wiki/Streuung_(Physik)\" title=\"Streuung (Physik)\">gestreut</a>, <a href=\"/wiki/Reflexion_(Physik)\" title=\"Reflexion (Physik)\">reflektiert</a>, <a href=\"/wiki/Brechung_(Physik)\" title=\"Brechung (Physik)\">gebrochen und verlangsamt</a> oder <a href=\"/wiki/Absorption_(Physik)\" title=\"Absorption (Physik)\">absorbiert</a> werden.\n</p><p>Licht ist der f\u00fcr das menschliche Auge ad\u00e4quate <a href=\"/wiki/Sinnesreiz\" class=\"mw-redirect\" title=\"Sinnesreiz\">Sinnesreiz</a>. Dabei wird die <a href=\"/wiki/Intensit%C3%A4t_(Physik)\" title=\"Intensit\u00e4t (Physik)\">Intensit\u00e4t</a> des Lichts als <a href=\"/wiki/Helligkeit\" title=\"Helligkeit\">Helligkeit</a> wahrgenommen, die spektrale Zusammensetzung als <a href=\"/wiki/Farbe\" title=\"Farbe\">Farbe</a>.<sup id=\"cite_ref-3\" class=\"reference\"><a href=\"#cite_note-3\"><span class=\"cite-bracket\">&#91;</span>3<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p>\n<meta property=\"mw:PageProp/toc\" />\n<div class=\"mw-heading mw-heading2\"><h2 id=\"Geschichte\">Geschichte</h2><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=1\" title=\"Abschnitt bearbeiten: Geschichte\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=1\" title=\"Quellcode des Abschnitts bearbeiten: Geschichte\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<figure class=\"mw-default-size mw-halign-left\" typeof=\"mw:File/Thumb\"><a href=\"/wiki/Datei:Light_Amplification_by_Stimulated_Emission_of_Radiation.jpg\" class=\"mw-file-description\"><img src=\"//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Light_Amplification_by_Stimulated_Emission_of_Radiation.jpg/250px-Light_Amplification_by_Stimulated_Emission_of_Radiation.jpg\" decoding=\"async\" width=\"250\" height=\"188\" class=\"mw-file-element\" srcset=\"//upload.wikimedia.org/wikipedia/commons/thumb/d/df/Light_Amplification_by_Stimulated_Emission_of_Radiation.jpg/500px-Light_Amplification_by_Stimulated_Emission_of_Radiation.jpg 1.5x\" data-file-width=\"4032\" data-file-height=\"3024\" /></a><figcaption>Verschiedenfarbige Laser</figcaption></figure>\n<p>Bis weit in die Neuzeit hinein war weitgehend unklar, was Licht tats\u00e4chlich ist. Man glaubte teilweise, dass die Helligkeit den Raum ohne Zeitverz\u00f6gerung ausf\u00fcllt. <a href=\"/wiki/Pythagoras\" title=\"Pythagoras\">Pythagoras</a> und <a href=\"/wiki/Euklid\" title=\"Euklid\">Euklid</a> waren der Auffassung, dass \u201ehei\u00dfe Sehstrahlen\u201c von den Augen ausgehen und von anderen Objekten zur\u00fcckgedr\u00e4ngt werden.<sup id=\"cite_ref-4\" class=\"reference\"><a href=\"#cite_note-4\"><span class=\"cite-bracket\">&#91;</span>4<span class=\"cite-bracket\">&#93;</span></a></sup><sup id=\"cite_ref-5\" class=\"reference\"><a href=\"#cite_note-5\"><span class=\"cite-bracket\">&#91;</span>5<span class=\"cite-bracket\">&#93;</span></a></sup> W\u00fcrde dies stimmen, m\u00fcsste der Mensch auch im Dunklen sehen k\u00f6nnen.<sup id=\"cite_ref-6\" class=\"reference\"><a href=\"#cite_note-6\"><span class=\"cite-bracket\">&#91;</span>6<span class=\"cite-bracket\">&#93;</span></a></sup> Es gab jedoch auch schon seit der Antike Vorstellungen, nach denen das Licht von der Lichtquelle mit endlicher Geschwindigkeit ausgesendet wird.\n</p><p><a href=\"/wiki/Galileo_Galilei\" title=\"Galileo Galilei\">Galileo Galilei</a> versuchte als einer der ersten, die <a href=\"/wiki/Lichtgeschwindigkeit\" title=\"Lichtgeschwindigkeit\">Ausbreitungsgeschwindigkeit des Lichts</a> ernsthaft zu messen, jedoch ohne Erfolg. Daf\u00fcr waren die ihm zur Verf\u00fcgung stehenden Mittel viel zu grob. Dies gelang erst <a href=\"/wiki/Ole_R%C3%B8mer\" title=\"Ole R\u00f8mer\">Ole R\u00f8mer</a> anhand von Beobachtungsdaten der <a href=\"/wiki/Liste_der_Jupitermonde\" title=\"Liste der Jupitermonde\">Jupitermonde</a> 1675, insbesondere des Mondes <a href=\"/wiki/Io_(Mond)\" title=\"Io (Mond)\">Io</a>. Zwar betrug die Abweichung seines Messwerts (ca. 2,1 \u00b7 10<sup>8</sup> m/s) vom tats\u00e4chlichen Wert rund 30&#160;%, die eigentliche Leistung R\u00f8mers bestand jedoch darin, nachzuweisen, dass sich das Licht mit endlicher Geschwindigkeit ausbreitet.<sup id=\"cite_ref-7\" class=\"reference\"><a href=\"#cite_note-7\"><span class=\"cite-bracket\">&#91;</span>7<span class=\"cite-bracket\">&#93;</span></a></sup> R\u00f8mers Messwert wurde im Laufe der folgenden 200 Jahre durch immer raffiniertere Verfahren (vor allem durch <a href=\"/wiki/Hippolyte_Fizeau\" title=\"Hippolyte Fizeau\">Hippolyte Fizeau</a> und <a href=\"/wiki/L%C3%A9on_Foucault\" title=\"L\u00e9on Foucault\">L\u00e9on Foucault</a>) mehr und mehr pr\u00e4zisiert. Die Natur des Lichts blieb jedoch weiter ungekl\u00e4rt. Im 17. Jahrhundert versuchte <a href=\"/wiki/Isaac_Newton\" title=\"Isaac Newton\">Isaac Newton</a> mit seiner <a href=\"/wiki/Korpuskeltheorie\" title=\"Korpuskeltheorie\">Korpuskeltheorie</a>, die Ausbreitung des Lichts durch die Bewegung von kleinen Teilchen zu erkl\u00e4ren. Damit konnte man zwar die <a href=\"/wiki/Reflexion_(Physik)\" title=\"Reflexion (Physik)\">Reflexion</a> verstehen, nicht jedoch manche andere optische Ph\u00e4nomene, wie die <a href=\"/wiki/Beugung_(Physik)\" title=\"Beugung (Physik)\">Beugung</a>, bei der es sich eindeutig um ein Wellenph\u00e4nomen handelt. Zur gleichen Zeit begr\u00fcndeten <a href=\"/wiki/Christiaan_Huygens\" title=\"Christiaan Huygens\">Christiaan Huygens</a> und andere die <a href=\"/wiki/Wellentheorie_des_Lichts\" class=\"mw-redirect\" title=\"Wellentheorie des Lichts\">Wellentheorie des Lichts</a>,<sup id=\"cite_ref-8\" class=\"reference\"><a href=\"#cite_note-8\"><span class=\"cite-bracket\">&#91;</span>8<span class=\"cite-bracket\">&#93;</span></a></sup> die sich aber erst Anfang des 19. Jahrhunderts nach den <a href=\"/wiki/Doppelspalt\" class=\"mw-redirect\" title=\"Doppelspalt\">Doppelspalt</a>&#173;experimenten von <a href=\"/wiki/Thomas_Young\" title=\"Thomas Young\">Thomas Young</a> zunehmend durchsetzte.<sup id=\"cite_ref-9\" class=\"reference\"><a href=\"#cite_note-9\"><span class=\"cite-bracket\">&#91;</span>9<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p><p><a href=\"/wiki/Michael_Faraday\" title=\"Michael Faraday\">Michael Faraday</a> erbrachte 1846 als erster den Nachweis, dass Licht und Magnetismus zwei miteinander verbundene physikalische Ph\u00e4nomene sind. Er ver\u00f6ffentlichte den von ihm gefundenen <a href=\"/wiki/Magnetooptik#Magnetooptische_Effekte\" title=\"Magnetooptik\">magnetooptischen Effekt</a>, der heute als <a href=\"/wiki/Faraday-Effekt\" title=\"Faraday-Effekt\">Faraday-Effekt</a><sup id=\"cite_ref-10\" class=\"reference\"><a href=\"#cite_note-10\"><span class=\"cite-bracket\">&#91;</span>10<span class=\"cite-bracket\">&#93;</span></a></sup> bezeichnet wird, unter dem Titel <i>\u00dcber die Magnetisierung des Lichts und die Belichtung der Magnetkraftlinien</i>.<sup id=\"cite_ref-11\" class=\"reference\"><a href=\"#cite_note-11\"><span class=\"cite-bracket\">&#91;</span>11<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p><p><a href=\"/wiki/James_Clerk_Maxwell\" title=\"James Clerk Maxwell\">James Clerk Maxwell</a> formulierte 1864 die noch heute g\u00fcltigen <a href=\"/wiki/Maxwell-Gleichungen\" title=\"Maxwell-Gleichungen\">Grundgleichungen</a> der Elektrodynamik und erkannte, dass dadurch die Existenz freier <a href=\"/wiki/Elektromagnetische_Welle\" title=\"Elektromagnetische Welle\">elektromagnetischer Wellen</a> vorhergesagt wurde. Da deren vorhergesagte Ausbreitungsgeschwindigkeit mit der bekannten Lichtgeschwindigkeit \u00fcbereinstimmte, schloss er, dass das Licht wohl eine elektromagnetische Welle sei.<sup id=\"cite_ref-12\" class=\"reference\"><a href=\"#cite_note-12\"><span class=\"cite-bracket\">&#91;</span>12<span class=\"cite-bracket\">&#93;</span></a></sup> Er vermutete (wie damals nahezu alle Physiker), dass diese Welle nicht im leeren Raum existieren k\u00f6nne, sondern ein Ausbreitungsmedium brauche. Dieses Medium, das das gesamte Weltall ausf\u00fcllen m\u00fcsste, wurde als <a href=\"/wiki/%C3%84ther_(Physik)\" title=\"\u00c4ther (Physik)\">\u00c4ther</a> bezeichnet.<sup id=\"cite_ref-13\" class=\"reference\"><a href=\"#cite_note-13\"><span class=\"cite-bracket\">&#91;</span>13<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p><p>Mit der darauf aufbauenden <i>elektromagnetischen Lichttheorie</i> schienen im ausgehenden 19. Jahrhundert beinahe alle Fragen zum Licht gekl\u00e4rt. Allerdings lie\u00df sich einerseits der postulierte \u00c4ther nicht nachweisen (siehe <a href=\"/wiki/Michelson-Morley-Experiment\" title=\"Michelson-Morley-Experiment\">Michelson-Morley-Experiment</a>), was letztendlich das Tor zur <a href=\"/wiki/Spezielle_Relativit%C3%A4tstheorie\" title=\"Spezielle Relativit\u00e4tstheorie\">speziellen Relativit\u00e4tstheorie</a> aufstie\u00df. Andererseits schien unter anderem der <a href=\"/wiki/Photoelektrischer_Effekt\" title=\"Photoelektrischer Effekt\">Photoeffekt</a> der Wellennatur des Lichts zu widersprechen. So entstand eine radikal neue Sichtweise des Lichts, die durch die Quantenhypothese von <a href=\"/wiki/Max_Planck\" title=\"Max Planck\">Max Planck</a> und <a href=\"/wiki/Albert_Einstein\" title=\"Albert Einstein\">Albert Einstein</a> begr\u00fcndet wurde. Kernpunkt dieser Hypothese ist der <a href=\"/wiki/Welle-Teilchen-Dualismus\" title=\"Welle-Teilchen-Dualismus\">Welle-Teilchen-Dualismus</a>, der das Licht nun nicht mehr ausschlie\u00dflich als Welle oder ausschlie\u00dflich als Teilchen beschreibt, sondern als <a href=\"/wiki/Quantenobjekt\" title=\"Quantenobjekt\">Quantenobjekt</a>.<sup id=\"cite_ref-14\" class=\"reference\"><a href=\"#cite_note-14\"><span class=\"cite-bracket\">&#91;</span>14<span class=\"cite-bracket\">&#93;</span></a></sup> Als solches vereint es Eigenschaften von Welle und von Teilchen, ohne das eine oder das andere zu sein und entzieht sich somit unserer konkreten Anschauung. Daraus entstand Anfang des 20. Jahrhunderts die <a href=\"/wiki/Quantenphysik\" title=\"Quantenphysik\">Quantenphysik</a> und sp\u00e4ter die <a href=\"/wiki/Quantenelektrodynamik\" title=\"Quantenelektrodynamik\">Quantenelektrodynamik</a>, die bis heute unser Verst\u00e4ndnis von der Natur des Lichts darstellt.<sup id=\"cite_ref-15\" class=\"reference\"><a href=\"#cite_note-15\"><span class=\"cite-bracket\">&#91;</span>15<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p>\n<div class=\"mw-heading mw-heading2\"><h2 id=\"Wissenschaft\">Wissenschaft</h2><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=2\" title=\"Abschnitt bearbeiten: Wissenschaft\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=2\" title=\"Quellcode des Abschnitts bearbeiten: Wissenschaft\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<div class=\"mw-heading mw-heading3\"><h3 id=\"Physik\">Physik</h3><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=3\" title=\"Abschnitt bearbeiten: Physik\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=3\" title=\"Quellcode des Abschnitts bearbeiten: Physik\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<p>Im Folgenden werden die wichtigsten Modelle zur Beschreibung des Lichts vorgestellt. Wie alle Modelle in der Physik sind auch die hier aufgef\u00fchrten in ihrem Geltungsbereich beschr\u00e4nkt. Das bedeutet, dass jeder <a href=\"/wiki/Strahlengang\" title=\"Strahlengang\">Strahlengang</a> auch dann allen optischen Gesetzen gen\u00fcgen w\u00fcrde, wenn man die Ausbreitungsrichtung des Lichts umkehren w\u00fcrde.</li></ul>\n<div class=\"mw-heading mw-heading5\"><h5 id=\"Reflexion\">Reflexion</h5><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=6\" title=\"Abschnitt bearbeiten: Reflexion\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=6\" title=\"Quellcode des Abschnitts bearbeiten: Reflexion\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<figure class=\"mw-default-size\" typeof=\"mw:File/Thumb\"><a href=\"/wiki/Datei:Reflexion.svg\" class=\"mw-file-description\"><img src=\"//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Reflexion.svg/250px-Reflexion.svg.png\" decoding=\"async\" width=\"250\" height=\"250\" class=\"mw-file-element\" srcset=\"//upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Reflexion.svg/375px-Reflexion.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Reflexion.svg/500px-Reflexion.svg.png 2x\" data-file-width=\"300\" data-file-height=\"300\" /></a><figcaption>Reflexion und Brechung an der Grenzschicht zweier transparenter Medien mit unterschiedlichem Brechungsindex</figcaption></figure>\n<p>Von spiegelnden Oberfl\u00e4chen (blankes Metall, Wasseroberfl\u00e4che) wird Licht nach dem <a href=\"/wiki/Reflexionsgesetz\" class=\"mw-redirect\" title=\"Reflexionsgesetz\">Reflexionsgesetz</a> reflektiert. In diesem Fall tritt statt der Brechung eine <a href=\"/wiki/Totalreflexion\" title=\"Totalreflexion\">Totalreflexion</a> auf.\n</p>\n<div class=\"mw-heading mw-heading4\"><h4 id=\"Wellenoptik\">Wellenoptik</h4><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=8\" title=\"Abschnitt bearbeiten: Wellenoptik\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=8\" title=\"Quellcode des Abschnitts bearbeiten: Wellenoptik\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<div class=\"hauptartikel\" role=\"navigation\"><span class=\"hauptartikel-pfeil\" title=\"siehe\" aria-hidden=\"true\" role=\"presentation\">\u2192&#160;</span><i><span class=\"hauptartikel-text\">Hauptartikel</span>: <a href=\"/wiki/Wellenoptik\" title=\"Wellenoptik\">Wellenoptik</a></i></div>\n<figure class=\"mw-default-size\" typeof=\"mw:File/Thumb\"><a href=\"/wiki/Datei:Doubleslit3Dspectrum.gif\" class=\"mw-file-description\"><img src=\"//upload.wikimedia.org/wikipedia/commons/thumb/3/33/Doubleslit3Dspectrum.gif/250px-Doubleslit3Dspectrum.gif\" decoding=\"async\" width=\"250\" height=\"256\" class=\"mw-file-element\" srcset=\"//upload.wikimedia.org/wikipedia/commons/3/33/Doubleslit3Dspectrum.gif 1.5x\" data-file-width=\"293\" data-file-height=\"300\" /></a><figcaption>Beugung einer ebenen Welle an einem Doppelspalt: Von den beiden Spalten geht je eine Elementarwelle aus, die beide zu dem typischen Beugungsmuster eines Doppelspalts interferieren.</figcaption></figure>\n<p>Der Wellenoptik liegt das <a href=\"/wiki/Huygenssches_Prinzip\" title=\"Huygenssches Prinzip\">Prinzip von Huygens und Fresnel</a> zugrunde.\n</p><p>Jeder Punkt einer Wellenfront ist der Ausgangspunkt einer Elementarwelle. Eine Wellenfront ergibt sich als \u00dcberlagerung dieser Elementarwellen.\n</p><p>Mit <i>Elementarwelle</i> ist in diesem Zusammenhang eine Kugelwelle gemeint, die von einem bestimmten Punkt ausgeht. <i>Wellenfronten</i> sind die Fl\u00e4chen gleicher Phase. Der Abstand zwischen zwei aufeinander folgenden Wellenfronten ist somit die Wellenl\u00e4nge. Die Wellenfronten einer ebenen Welle sind also Ebenen, die Wellenfronten von Elementarwellen sind konzentrische Kugelfl\u00e4chen. Die Ausbreitungsrichtung (also die Richtung des Wellenvektors) bildet stets eine <a href=\"/wiki/Normale\" class=\"mw-redirect\" title=\"Normale\">Normale</a> zur Wellenfront. Mit der Wellenoptik lassen sich alle Ph\u00e4nomene der Beugung und <a href=\"/wiki/Interferenz_(Physik)\" title=\"Interferenz (Physik)\">Interferenz</a> verstehen. Sie eignet sich aber auch, das Reflexions- und das Brechungsgesetz herzuleiten. Die Wellenoptik widerspricht also nicht der Strahlenoptik, sondern erweitert und vertieft diese.\n</p><p>Historisch nimmt die Wellenoptik von Huygens und Fresnel schon eine wesentliche Erkenntnis der Elektrodynamik vorweg: Lichtwellen sind elektromagnetische Wellen.\n</p>\n<div class=\"mw-heading mw-heading4\"><h4 id=\"Photonen\">Photonen</h4><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=9\" title=\"Abschnitt bearbeiten: Photonen\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=9\" title=\"Quellcode des Abschnitts bearbeiten: Photonen\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<div class=\"hauptartikel\" role=\"navigation\"><span class=\"hauptartikel-pfeil\" title=\"siehe\" aria-hidden=\"true\" role=\"presentation\">\u2192&#160;</span><i><span class=\"hauptartikel-text\">Hauptartikel</span>: <a href=\"/wiki/Photon\" title=\"Photon\">Photon</a></i></div>\n<p>In der Quantenphysik wird Licht nicht mehr als klassische Welle, sondern als Quantenobjekt aufgefasst. Es ist also \u201ez\u00e4hlbar\u201c wie ein Teilchen. Trotzdem bleibt alles, was hier bisher \u00fcber die Welleneigenschaften des Lichts gesagt wurde, g\u00fcltig. Je nach Betrachtungsweise zeigen sie Eigenschaften der einen oder der anderen.\n</p><p>In der heute g\u00e4ngigsten Interpretation der Quantenmechanik (<a href=\"/wiki/Kopenhagener_Deutung\" class=\"mw-redirect\" title=\"Kopenhagener Deutung\">Kopenhagener Deutung</a>) kann man den genauen Ort eines Photons nicht <i>a priori</i> vorhersagen. Man kann nur Aussagen \u00fcber die Wahrscheinlichkeit machen, mit der ein Photon an einer bestimmten Stelle auftreffen wird. Diese Wahrscheinlichkeitsdichte ist durch das <a href=\"/wiki/Betragsquadrat\" title=\"Betragsquadrat\">Betragsquadrat</a> der Amplitude der Lichtwelle gegeben.\n</p><p>Historisch wurde die quantenmechanische Beschreibung des Lichts notwendig, weil sich einige Ph\u00e4nomene mit der rein <a href=\"/wiki/Klassische_Physik\" title=\"Klassische Physik\">klassischen</a> Elektrodynamik nicht erkl\u00e4ren lie\u00dfen:\n</p>\n<ul><li>Stellt man sich eine thermische Lichtquelle (im Idealfall ein <a href=\"/wiki/Schwarzer_K%C3%B6rper\" title=\"Schwarzer K\u00f6rper\">Schwarzer K\u00f6rper</a>) als eine Ansammlungen von vielen atomaren Oszillatoren vor, die mit dem Strahlungsfeld im Gleichgewicht stehen, so w\u00fcrde eine klassische Herleitung zur \u201eUV-Katastrophe\u201c f\u00fchren, kurzwellige Strahlung m\u00fcsste im Spektrum des Schwarzen K\u00f6rpers viel st\u00e4rker vertreten sein, als sie es ist. Die von den <a href=\"/wiki/Chlorophyll\" class=\"mw-redirect\" title=\"Chlorophyll\">Chlorophyll</a>-Molek\u00fclen in den <a href=\"/wiki/Chloroplasten\" class=\"mw-redirect\" title=\"Chloroplasten\">Chloroplasten</a> absorbierte Lichtenergie wird genutzt, um Wassermolek\u00fcle zu spalten (<a href=\"/wiki/Photolyse\" title=\"Photolyse\">Photolyse</a>) und so <a href=\"/wiki/Reduktionsmittel\" title=\"Reduktionsmittel\">Reduktionsmittel</a> f\u00fcr die Photosynthese herzustellen. Diese werden in einem zweiten Schritt verwendet, um <a href=\"/wiki/Kohlenstoffdioxid\" title=\"Kohlenstoffdioxid\">Kohlenstoffdioxid</a> schrittweise schlie\u00dflich zu <a href=\"/wiki/Glucose\" title=\"Glucose\">Glucose</a> zu reduzieren, woraus unter anderem <a href=\"/wiki/St%C3%A4rke\" title=\"St\u00e4rke\">St\u00e4rke</a> aufgebaut wird. Der bei der Fotolyse anfallende Sauerstoff wird als Reststoff an die Atmosph\u00e4re abgegeben. Kohlenstoffdioxid-<a href=\"/wiki/Assimilation_(Biologie)\" title=\"Assimilation (Biologie)\">Assimilation</a>. Organismen, die mithilfe von Licht dazu in der Lage sind, nennt man <i>photo</i>-<a href=\"/wiki/Autotrophie\" title=\"Autotrophie\">autotroph</a>. Neben den <a href=\"/wiki/Gef%C3%A4%C3%9Fpflanzen\" title=\"Gef\u00e4\u00dfpflanzen\">Gef\u00e4\u00dfpflanzen</a> geh\u00f6ren auch <a href=\"/wiki/Moose\" title=\"Moose\">Moose</a>, <a href=\"/wiki/Algen\" class=\"mw-redirect\" title=\"Algen\">Algen</a> und einige <a href=\"/wiki/Bakterien\" title=\"Bakterien\">Bakterien</a> dazu, beispielsweise <a href=\"/wiki/Cyanobakterien\" title=\"Cyanobakterien\">Cyanobakterien</a> und <a href=\"/wiki/Purpurbakterien\" title=\"Purpurbakterien\">Purpurbakterien</a>. Alle <a href=\"/wiki/Heterotrophie\" title=\"Heterotrophie\">heterotrophen</a> Organismen sind von dieser Assimilation abh\u00e4ngig, weil sie ihren Energiebedarf nur aus organischen Verbindungen, die sie mit der Nahrung aufnehmen m\u00fcssen, decken k\u00f6nnen.\n</p><p>Die Konkurrenz der Pflanzen ums Licht macht sich im \u201eStockwerkaufbau\u201c des <a href=\"/wiki/Wald\" title=\"Wald\">Waldes</a> und der damit verbundenen Spezialisierung von Licht- und <a href=\"/wiki/Schattenpflanze\" title=\"Schattenpflanze\">Schattenpflanzen</a> oder in der jahrzeitlichen Abfolge verschiedener <a href=\"/wiki/Aspekt_(Biologie)\" title=\"Aspekt (Biologie)\">Aspekte</a> bemerkbar. In Gew\u00e4ssern dient <i>nur</i> die lichtdurchflutete oberste Schicht, die <a href=\"/wiki/Trophogene_Zone\" title=\"Trophogene Zone\">N\u00e4hrschicht</a>, der Bildung von Biomasse und Sauerstoff, haupts\u00e4chlich durch <a href=\"/wiki/Phytoplankton\" title=\"Phytoplankton\">Phytoplankton</a>. Weil viele Tiere und Einzeller durch das hohe Nahrungsangebot und den vergleichsweise hohen Sauerstoffgehalt des Wassers hier gute Lebensbedingungen finden, werden sie durch das Licht angelockt.\n</p>\n<figure class=\"mw-default-size\" typeof=\"mw:File/Thumb\"><a href=\"/wiki/Datei:Lampyris_noctiluca.jpg\" class=\"mw-file-description\"><img src=\"//upload.wikimedia.org/wikipedia/commons/thumb/2/2b/Lampyris_noctiluca.jpg/250px-Lampyris_noctiluca.jpg\" decoding=\"async\" width=\"190\" height=\"229\" class=\"mw-file-element\" srcset=\"//upload.wikimedia.org/wikipedia/commons/thumb/2/2b/Lampyris_noctiluca.jpg/330px-Lampyris_noctiluca.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/2/2b/Lampyris_noctiluca.jpg/380px-Lampyris_noctiluca.jpg 2x\" data-file-width=\"408\" data-file-height=\"492\" /></a><figcaption><a href=\"/wiki/Gro%C3%9Fer_Leuchtk%C3%A4fer\" title=\"Gro\u00dfer Leuchtk\u00e4fer\">Gro\u00dfer Leuchtk\u00e4fer</a> (<i>Lampyris noctiluca</i>), weibliches Tier beim hochsommerlichen Lock-Leuchten</figcaption></figure>\n<p>Der Licht- oder Sehsinn ist f\u00fcr viele Tiere einer der wichtigsten Sinne. Er dient zur Orientierung im Raum, zur Steuerung des <a href=\"/wiki/Circadiane_Rhythmik\" title=\"Circadiane Rhythmik\">Tag-Nacht-Rhythmus</a>, zum Erkennen von Gefahren, zum Aufsp\u00fcren von Beute und zur Kommunikation mit Artgenossen. Daher haben sich im Laufe der Evolution in den verschiedensten <a href=\"/wiki/Taxon\" title=\"Taxon\">Taxa</a> die unterschiedlichsten Lichtsinnesorgane entwickelt. Diese reichen von den einfachen Augenflecken von <i><a href=\"/wiki/Euglena\" class=\"mw-redirect\" title=\"Euglena\">Euglena</a></i> \u00fcber einfache Pigmentfelder bis zu den komplex aufgebauten <a href=\"/wiki/Facettenauge\" title=\"Facettenauge\">Facettenaugen</a> und <a href=\"/wiki/Linsenauge\" class=\"mw-redirect\" title=\"Linsenauge\">Linsenaugen</a>. Nur wenige Tiere sind vollkommen unempfindlich f\u00fcr Lichtreize. Dies ist h\u00f6chstens dann der Fall, wenn sie in v\u00f6lliger Dunkelheit leben, wie H\u00f6hlentiere.\n</p><p>Sowohl f\u00fcr R\u00e4uber- als auch Beutetiere ist es von Vorteil, <i>nicht</i> gesehen zu werden. Anpassungen daran sind <a href=\"/wiki/Tarnung_(Biologie)\" title=\"Tarnung (Biologie)\">Tarnung</a> und <a href=\"/wiki/Temporale_Spezialisten\" class=\"mw-redirect\" title=\"Temporale Spezialisten\">Nachtaktivit\u00e4t</a>. Erstaunlicherweise haben dahingegen viele Lebewesen selbst die F\u00e4higkeit entwickelt zu leuchten. Das bekannteste Beispiel ist der <a href=\"/wiki/Leuchtk%C3%A4fer\" title=\"Leuchtk\u00e4fer\">Leuchtk\u00e4fer</a>. Man findet dieses Ph\u00e4nomen der <a href=\"/wiki/Biolumineszenz\" title=\"Biolumineszenz\">Biolumineszenz</a> aber auch bei <a href=\"/wiki/Tiefseefische\" class=\"mw-redirect\" title=\"Tiefseefische\">Tiefseefischen</a>, <a href=\"/wiki/Leuchtkrebs\" class=\"mw-redirect\" title=\"Leuchtkrebs\">Leuchtkrebsen</a>, Pilzen (<a href=\"/wiki/Hallimasch\" class=\"mw-redirect\" title=\"Hallimasch\">Hallimasch</a>) oder Bakterien. Der Nutzen der Biolumineszenz wird vor allem mit innerartlicher Kommunikation, der Abschreckung von Fra\u00dffeinden und dem Anlocken von Beute erkl\u00e4rt.\n</p>\n<div class=\"mw-heading mw-heading4\"><h4 id=\"Licht_als_Sinnesreiz\">Licht als Sinnesreiz</h4><span class=\"mw-editsection\"><span class=\"mw-editsection-bracket\">[</span><a href=\"/w/index.php?title=Licht&amp;veaction=edit&amp;section=15\" title=\"Abschnitt bearbeiten: Licht als Sinnesreiz\" class=\"mw-editsection-visualeditor\"><span>Bearbeiten</span></a><span class=\"mw-editsection-divider\"> | </span><a href=\"/w/index.php?title=Licht&amp;action=edit&amp;section=15\" title=\"Quellcode des Abschnitts bearbeiten: Licht als Sinnesreiz\"><span>Quelltext bearbeiten</span></a><span class=\"mw-editsection-bracket\">]</span></span></div>\n<figure class=\"mw-default-size\" typeof=\"mw:File/Thumb\"><a href=\"/wiki/Datei:Eye_scheme.svg\" class=\"mw-file-description\"><img src=\"//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Eye_scheme.svg/330px-Eye_scheme.svg.png\" decoding=\"async\" width=\"270\" height=\"223\" class=\"mw-file-element\" srcset=\"//upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Eye_scheme.svg/500px-Eye_scheme.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Eye_scheme.svg/540px-Eye_scheme.svg.png 2x\" data-file-width=\"632\" data-file-height=\"522\" /></a><figcaption>Schematischer L\u00e4ngsschnitt durch das menschliche Auge</figcaption></figure>\n<p>Das Licht, das ins menschliche Auge f\u00e4llt, wird durch den Brechapparat (bestehend aus <a href=\"/wiki/Hornhaut\" title=\"Hornhaut\">Hornhaut</a>, vorderer und hinterer Augenkammer, <a href=\"/wiki/Linse_(Auge)\" title=\"Linse (Auge)\">Linse</a> und <a href=\"/wiki/Glask%C3%B6rper\" title=\"Glask\u00f6rper\">Glask\u00f6rper</a>) auf die <a href=\"/wiki/Netzhaut\" title=\"Netzhaut\">Netzhaut</a> projiziert. Dort entsteht ein reelles, auf dem <a href=\"/wiki/Kopf\" title=\"Kopf\">Kopf</a> stehendes Bild (vergleichbar dem Vorgang in einer Fotokamera). Dadurch werden die in der Netzhaut befindlichen <a href=\"/wiki/Photorezeptor\" class=\"mw-redirect\" title=\"Photorezeptor\">Photorezeptoren</a> (= Lichtsinneszellen) gereizt, die den Reiz in ein elektrisches Signal wandeln. Dieses Signal wird \u00fcber den Sehnerv, in den die einzelnen Nervenstr\u00e4nge der Netzhaut m\u00fcnden, zum <a href=\"/wiki/Gehirn\" title=\"Gehirn\">Gehirn</a> geleitet. Dort werden die auf dem Kopf stehenden Bilder unserer Umwelt dann in Echtzeit \u201egerade ger\u00fcckt.\u201c\n</p><p>Lichtintensit\u00e4t wird als <i>Helligkeit</i> empfunden. Das Auge kann sich durch verschiedene Mechanismen an die \u2013 viele Zehnerpotenzen umfassenden \u2013 Intensit\u00e4ten anpassen (siehe <a href=\"/wiki/Adaptation_(Auge)\" title=\"Adaptation (Auge)\">Adaption</a>). Die empfundene Helligkeit h\u00e4ngt dabei mit der tats\u00e4chlichen Intensit\u00e4t \u00fcber das <a href=\"/wiki/Weber-Fechner-Gesetz\" title=\"Weber-Fechner-Gesetz\">Weber-Fechner-Gesetz</a> zusammen.\n</p><p>Die spektrale Zusammensetzung des Lichtreizes wird als <a href=\"/wiki/Farbe\" title=\"Farbe\">Farbe</a> wahrgenommen, wobei das menschliche Auge Licht mit Wellenl\u00e4ngen zwischen ca. 380 nm und 750 nm erfassen kann. Trennt man wei\u00dfes Licht (durch ein Prisma) auf, so erscheinen die Wellenl\u00e4ngen als Farben des Regenbogens.\n</p>\n<table class=\"wikitable float-right\" style=\"text-align:center; Die <a href=\"/wiki/Zapfen_(Auge)\" title=\"Zapfen (Auge)\">Zapfen</a> hingegen, die an st\u00e4rkere Intensit\u00e4ten angepasst sind, kommen in drei verschiedenen Typen vor, die jeweils bei einer anderen Wellenl\u00e4nge ihr Reaktionsoptimum haben. Der <a href=\"/wiki/Ligand\" title=\"Ligand\">Ligand</a> <a href=\"/wiki/Retinal\" title=\"Retinal\">Retinal</a> macht dabei eine <a href=\"/wiki/Isomerisierung\" title=\"Isomerisierung\">Isomerisierung</a> durch, die dazu f\u00fchrt, dass das Rhodopsin zerf\u00e4llt und die Signalkaskade der <a href=\"/wiki/Phototransduktion\" title=\"Phototransduktion\">Phototransduktion</a> in Gang setzt. Die dadurch verursachte <a href=\"/wiki/Hyperpolarisation_(Biologie)\" title=\"Hyperpolarisation (Biologie)\">Hyperpolarisation</a> der Zellmembran der St\u00e4bchen und Zapfen bewirkt ein elektrisches Signal, das an die nachgeschalteten Nervenzellen weitergegeben wird. </p><div class=\"sieheauch\" role=\"navigation\" style=\"font-style:italic;\"><span class=\"sieheauch-text\">Siehe auch</span>: <a href=\"/wiki/Sehbahn\" title=\"Sehbahn\">Sehbahn</a></div>\n<p>Neben Zapfen und St\u00e4bchen gibt es einen dritten Lichtrezeptor, die <a href=\"/wiki/Melanopsin\" title=\"Melanopsin\">Melanopsin</a> enthaltende <a href=\"/wiki/Photosensitive_retinale_Ganglienzelle\" title=\"Photosensitive retinale Ganglienzelle\">photosensitive retinale Ganglienzelle</a>. Ihre Entdeckung Anfang der Jahrtausendwende forcierte die Entwicklung von tageslicht\u00e4hnlichen Beleuchtungskonzepten f\u00fcr Innenr\u00e4ume, wie bspw. das <a href=\"/wiki/Human_Centric_Lighting\" title=\"Human Centric Lighting\">Human Centric Lighting</a>.<sup id=\"cite_ref-20\" class=\"reference\"><a href=\"#cite_note-20\"><span class=\"cite-bracket\">&#91;</span>20<span class=\"cite-bracket\">&#93;</span></a></sup>\n</p><p>Die Leistungen der Lichtsinnesorgane anderer Lebewesen unterscheiden sich zum Teil erheblich von denen des Menschen.
